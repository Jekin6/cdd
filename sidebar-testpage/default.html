Get Started

  • Introduction

    • This guide is designed to instruct developers on the deployment of the ChemDraw Direct drawing tool. It describes the procedures for loading and attaching it, as well as API usage and functionality.

  • Loading the ChemDraw Direct Module

    • To gain access to the classes you need to use ChemDraw Direct, you first need to load its module. To do this, you add the required <script> tag: 

    <script type="text/javascript" 
            src="chemdrawweb/chemdrawweb.nocache.js"></script>
    Once this script tag is evaluated, the loading of the module will begin. The loading of the module is asynchronous and the classes in the module will not be available until the loading is complete. To avoid unexpected page scale, you need to add the following metadata in the header of the html file. 
    <meta name="viewport" content="width=device-width, initial-scale=1.0, 
minimum-scale=1.0, maximum-scale=1.0, user-scalable=no" />

  • Using the ChemdrawWebManager

    • A JavaScript function is provided to manage the asynchronous loading of the module, for ease-of-use. To use the Javascript function, include the following <script> tag: 

    <script type="text/javascript" 
            src="chemdrawweb/ChemdrawWebManager.js"></script>
    Once this tag is evaluated, a new function becomes available you can use to attach ChemDraw in a single line. The function is used as: 
    perkinelmer.ChemdrawWebManager.attach({id: "cdwAnchor", callback: toolAttached});
    The attach function may be used to attach several instances of ChemDraw to the DOM and may be called at any time. Additional parameters to the attach method's request object are defined in ChemdrawWebManager.js.

  • Manually Adding ChemDraw Direct to the DOM

    • If you prefer fine-grained control over when and how ChemDraw is added to the DOM, you can forgo the usage of ChemDrawWebManager (in section 3 above) and do it yourself. To ensure the classes are available, you must use them within a function attached to the window called chemdrawLoaded.
    Example:

    <script>
    function chemdrawLoaded() {
        var cdw = new perkinelmer.ChemdrawWeb();
        cdw.attach("cdwAnchor");
 
       ...
 
    }
</script>
    1. Constructing a ChemDraw Direct Object

      2. The construction of a ChemDraw Direct instance is the same as creating any other JavaScript object.
      Example:

      <script>
    function chemdrawLoaded() {
    var cdw = new perkinelmer.ChemdrawWeb();
      cdw.attach("cdwAnchor");
 
        ...   
 
    }
</script>

    2. Attaching ChemDraw Direct to the DOM Tree

      3. ChemDraw must be attached to an element on the DOM tree for you to view it. This is accomplished by passing the ID of an element into the ChemDraw's attach method. Note: The element ChemDraw is attached to must be empty. You can also pass an element object into the ChemDraw's attachToElement function.
      Example with id:

      
<body>
  <div id="cdwAnchor"></div>
  <script>
    function chemdrawLoaded() {
    var cdw = new perkinelmer.ChemdrawWeb();
      cdw.attach("cdwAnchor");
 
        ...   
 
    }
  </script>
</body>
      Example with element:
      
<body>
  <div id="cdwAnchor"></div>
  <script>
    function chemdrawLoaded() {
        var element = document.getElementById("cdwAnchor");
        var cdw = new perkinelmer.ChemdrawWeb();
        cdw.attachToElement(element);
 
        ...   
 
    }
  </script>
</body>
